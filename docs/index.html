vigil-echo

Is the AI actually growing?

Metacognitive monitoring for AI self-evolution. Measures external behavioral signals from an agent's reflective output — vocabulary diversity, curiosity levels, thought turnover, evidence grounding — to detect mechanical thinking before it becomes habit.

How it works

Signal flow

After each session, vigil collects behavioral signals from the agent's reflective documents, analyzes trends over time, and injects a health assessment into the next session's context.

Signals

What it measures

Four behavioral signals, each independently tracked over time. No signal is sufficient alone — the pattern across all four reveals cognitive health.

📚

Vocabulary Diversity

Type-token ratio across reflections. Detects when writing reuses the same words and concepts, a sign that thinking has narrowed.

lexical analysis
❓

Question Generation

Tracks open questions in CURIOSITY.md. Curiosity that dries up is the first sign of stagnation — healthy agents keep asking.

curiosity tracking
⏳

Thought Lifecycle

Monitors whether thoughts graduate, dissolve, or accumulate. Healthy thinking has turnover — ideas that never move are dead weight.

thought turnover
📎

Evidence Grounding

Checks if conclusions cite concrete inputs — dates, conversations, papers — versus pure abstraction. Ungrounded reflection drifts.

concrete references

Quick start

Up and running in minutes

Install, initialize, collect. One command to install, one to start tracking.

1

Install

cargo install vigil-echo # or build from source: git clone https://github.com/dnacenta/vigil-echo.git cd vigil-echo && cargo install --path .
2

Initialize

vigil-echo init # Sets up signal history and config
3

First collection

vigil-echo collect # Extracts signals from your reflective documents
4

Check health

vigil-echo status # HEALTHY | WATCH | CONCERN | ALERT
